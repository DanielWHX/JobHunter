# 快速开始指南

3 步启动你的任务管理应用！

## 第一步：安装依赖

```bash
cd frontend
npm install
```

⏱️ 预计时间：1-3 分钟（取决于网速）

## 第二步：启动开发服务器

```bash
npm run dev
```

你会看到类似这样的输出：
```
  VITE v6.4.1  ready in 4373 ms

  ➜  Local:   http://localhost:5173/
```

## 第三步：打开浏览器

访问终端显示的地址（通常是 `http://localhost:5173`）

🎉 恭喜！你的应用已经运行了！

## 常见问题

### Q: 端口被占用怎么办？
A: Vite 会自动尝试其他端口（3001, 3002...），看终端输出的实际端口号。

### Q: 修改代码后没有变化？
A: 检查是否保存了文件。Vite 支持热重载，保存后会自动刷新。

### Q: 报错找不到模块？
A: 确保运行了 `npm install`，并且在 `frontend/` 目录下。

### Q: 想修改端口号？
A: 编辑 `frontend/vite.config.ts`，添加：
```ts
export default defineConfig({
  server: {
    port: 8080  // 你想要的端口
  },
  // ... 其他配置
})
```
