# 🎯 JobHunter 启动指南

## 📱 现在就能用!

你的任务管理系统已经可以使用了!

### 第一步:启动项目

打开命令行,执行:

```bash
cd frontend
npm install
npm run dev
```

### 第二步:打开浏览器

访问终端显示的地址,例如:
- http://localhost:3000
- http://localhost:3001
- http://localhost:3002

(Vite 会自动选择可用端口)

### 第三步:开始使用

你会看到这样的界面:


## 🎮 如何使用

### 创建任务

1. 在左侧表单中输入**任务标题**(必填)
2. 选择**分类**:
   - 🏠 生活琐事
   - 📚 学校事情
   - 📝 身份事情
   - 💼 找工学习
   - 💪 健康运动
   - 🌱 个人发展
   - 📦 其他

3. 选择**优先级**(四象限法则):
   - 🔴 紧急且重要 - 立即做
   - 🔵 重要不紧急 - 安排时间
   - 🟡 紧急不重要 - 委托他人
   - ⚪ 不紧急不重要 - 有空再做

4. (可选)填写描述和预计时长
5. 点击"创建任务"

### 管理任务

- ☑️ **完成任务**: 点击任务前的复选框
- ✏️ **编辑任务**: 点击蓝色编辑图标
- 🗑️ **删除任务**: 点击红色删除图标

### 查看统计

右上角的统计卡片实时显示:
- 总任务数
- 待完成数量
- 已完成数量
- 完成率百分比

## ⚠️ 重要提示

### 数据会丢失!

当前数据存储在浏览器内存中,刷新页面后会丢失。

**解决方案**: 下次更新会添加 LocalStorage 支持

### 如何保存数据

目前的临时方案:
1. 完成任务后尽量不要刷新页面
2. 重要任务记录在其他地方备份
3. 等待 LocalStorage 功能更新

## 🐛 遇到问题?

### 端口被占用

**现象**:
```
Port 3000 is in use, trying another one...
Local: http://localhost:3001/
```

**解决**: 使用提示的新端口访问即可

### 页面空白

**检查**:
1. F12 打开控制台查看错误
2. 确认开发服务器正在运行
3. 清除缓存: Ctrl + Shift + R

### 安装失败

**解决**:
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```
