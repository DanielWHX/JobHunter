# 如何使用任务管理系统

## 🎯 功能概览

这个任务管理系统基于**艾森豪威尔矩阵**(四象限法则),帮助你高效管理每日任务。

### 四象限法则

```
        紧急                    不紧急
    ┌──────────────┬──────────────┐
重  │   第一象限    │   第二象限    │
要  │ 紧急且重要    │ 重要不紧急    │
    │  🔴 立即处理  │  🔵 计划安排  │
    ├──────────────┼──────────────┤
不  │   第三象限    │   第四象限    │
重  │ 紧急不重要    │ 不紧急不重要  │
要  │  🟡 委托他人  │  ⚪ 稍后处理  │
    └──────────────┴──────────────┘
```

## ✨ 基本操作

### 1. 创建新任务

**步骤**:
1. 在左侧表单中输入**任务标题**(必填)
2. 选择**任务分类**:
   - 🏠 生活琐事 - 日常生活杂事
   - 📚 学校事情 - 学习、作业相关
   - 📝 身份事情 - 证件、签证等
   - 💼 找工学习 - 求职、技能学习
   - 💪 健康运动 - 锻炼、体检
   - 🌱 个人发展 - 兴趣、成长
   - 📦 其他
3. 选择**优先级**(重要性和紧急程度)
4. (可选)填写描述和预计时长
5. 点击"创建任务"

**示例**:
```
标题: 完成 React 作业
描述: 第三章的练习题,包括组件和 hooks
分类: 学校事情
优先级: 紧急且重要
预计时长: 120 分钟
```

### 2. 完成任务

点击任务前的 ☑️ 复选框,任务会:
- 标题显示删除线
- 移动到"已完成"分组
- 统计数据自动更新

### 3. 编辑任务

1. 点击任务卡片上的 ✏️ 编辑图标
2. 左侧表单会自动填充任务信息
3. 修改后点击"更新任务"
4. 点击"取消"可放弃修改

### 4. 删除任务

点击任务卡片上的 🗑️ 删除图标,任务会立即删除。

⚠️ **注意**: 删除操作不可撤销!

## 💡 使用技巧

### 1. 如何确定优先级?

使用这个简单的决策树:

```
这件事很重要吗?
  ├─ 是 → 有截止日期/紧急吗?
  │        ├─ 是 → 🔴 紧急且重要(立即做)
  │        └─ 否 → 🔵 重要不紧急(安排时间)
  │
  └─ 否 → 有人在等你吗?
           ├─ 是 → 🟡 紧急不重要(委托/快速处理)
           └─ 否 → ⚪ 不紧急不重要(有空再做)
```

### 2. 每日最佳实践

**早上(8:00-9:00)**:
1. 查看所有待办任务
2. 确认优先级是否需要调整
3. 先处理 🔴 紧急且重要的任务

**中午(12:00-13:00)**:
1. 查看上午完成情况
2. 调整下午计划

**晚上(21:00-22:00)**:
1. 完成所有已完成的任务
2. 规划明天的任务
3. 删除不必要的任务

### 3. 时间估算技巧

**新手倾向低估时间**,建议:
- 估算时间 × 1.5 = 实际时间
- 留出缓冲时间
- 记录实际耗时,改进估算

**示例**:
```
任务: 写一篇博客
估算: 60 分钟
建议: 90 分钟(60 × 1.5)
```

### 4. 分解大任务

如果一个任务超过 2 小时,考虑分解:

**❌ 不好的任务**:
```
标题: 准备期末考试
预计: 10 小时
```

**✅ 好的任务**:
```
任务1: 整理数学笔记 - 2小时
任务2: 做物理练习题 - 2小时
任务3: 复习英语单词 - 1小时
任务4: 总结历史重点 - 2小时
```

## 📊 进度追踪

### 每日完成率目标

- 🥇 **优秀**: 完成率 ≥ 80%
- 🥈 **良好**: 完成率 60-80%
- 🥉 **及格**: 完成率 40-60%
- ⚠️ **需改进**: 完成率 < 40%

### 分析和改进

**如果完成率低**:
1. 是否创建了太多任务?
2. 时间估算是否准确?
3. 优先级是否合理?
4. 是否有太多干扰?

**优化建议**:
- 每天任务数控制在 5-8 个
- 专注完成高优先级任务
- 减少多任务切换

## ⚠️ 注意事项

### 当前限制

1. **数据不持久化**: 刷新页面后数据会丢失
   - 解决方案: 即将添加 LocalStorage 支持

2. **无法跨设备同步**: 数据只在本地浏览器
   - 解决方案: 后续会添加云端同步

3. **无截止日期提醒**: 需要手动检查
   - 解决方案: 计划添加通知功能

### 最佳实践

✅ **推荐做法**:
- 每天早上规划任务
- 及时更新任务状态
- 每天晚上复盘
- 合理估算时间
- 任务具体明确

❌ **避免做法**:
- 创建过于模糊的任务
- 不切实际的时间估算
- 过度创建低优先级任务
- 忘记标记完成状态
