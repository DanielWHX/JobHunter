# 前端学习路线

从零开始学习 React，通过这个项目掌握现代前端开发。

## 学习目标

通过这个项目，你将学会：
- ✅ React 基础概念和组件开发
- ✅ TypeScript 类型系统
- ✅ React Hooks 的使用
- ✅ 状态管理
- ✅ Tailwind CSS 样式开发
- ✅ 浏览器本地存储

## 第二周：React Hooks 深入

### Day 8-10: useState Hook

**学习内容**:
- `useState` 的工作原理
- 状态更新和重新渲染
- 数组和对象状态的更新

**代码示例**:
```tsx
// 在 App.tsx 中
const [showCompleted, setShowCompleted] = useState(true);

// 切换按钮
<button onClick={() => setShowCompleted(!showCompleted)}>
  {showCompleted ? '隐藏' : '显示'}已完成任务
</button>

// 条件过滤
const visibleTasks = showCompleted
  ? tasks
  : tasks.filter(t => t.status !== TaskStatus.COMPLETED);
```

**练习**:
1. 添加一个「显示/隐藏已完成任务」的切换按钮
2. 添加任务排序功能（按创建时间、优先级）

### Day 11-12: useEffect Hook

**阅读文件**: [hooks/useLocalStorage.ts](../frontend/src/hooks/useLocalStorage.ts)

**学习内容**:
- `useEffect` 的执行时机
- 依赖数组的作用
- 副作用清理

**代码示例**:
```tsx
// 统计使用时长
useEffect(() => {
  const startTime = Date.now();

  return () => {
    const duration = Date.now() - startTime;
    console.log(`使用时长: ${duration}ms`);
  };
}, []);
```

**练习**:
1. 添加一个计时器，显示页面打开时长
2. 当任务列表变化时，在控制台打印日志
3. 监听窗口大小变化，显示当前屏幕宽度

### Day 13-14: 自定义 Hooks

**学习内容**:
- 为什么需要自定义 Hooks？
- 如何提取可复用逻辑
- Hook 的命名规则（use 开头）

**练习**:
1. 创建 `useFilter` Hook
   ```tsx
   function useFilter<T>(items: T[], filterFn: (item: T) => boolean) {
     const [isActive, setIsActive] = useState(false);
     const filtered = isActive ? items.filter(filterFn) : items;
     return { filtered, isActive, toggle: () => setIsActive(!isActive) };
   }
   ```

2. 创建 `useDebounce` Hook（搜索防抖）

## 学习建议

1. **每天写代码**: 即使只有 30 分钟
2. **不要跳过练习**: 动手实践最重要
3. **遇到问题先思考**:
   - 检查控制台错误
   - 阅读错误信息
   - Google 搜索错误
   - 查看文档
4. **记笔记**: 记录学到的新概念
5. **不要追求完美**: 先让代码运行，再优化
