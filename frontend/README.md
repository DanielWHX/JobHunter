# JobHunter 前端 - 每日任务管理

基于 Vite + React + TypeScript + TailwindCSS 构建的现代化任务管理应用。

## 功能特性

### ✅ 已实现功能
- **任务创建与编辑**: 完整的表单支持
- **优先级管理**: 按照四象限法则分类
  - 紧急且重要
  - 重要不紧急
  - 紧急不重要
  - 不紧急不重要
- **任务分类**:
  - 生活琐事
  - 学校事情
  - 身份事情
  - 找工学习
  - 健康运动
  - 个人发展
  - 其他
- **任务状态跟踪**: 待处理、进行中、已完成、已取消
- **实时统计**: 总任务数、待完成、已完成、完成率
- **响应式设计**: 支持桌面和移动设备
- **深色模式**: 自动适配系统主题

## 快速开始

### 启动开发服务器

```bash
cd frontend
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
npm run build
npm run preview
```

## 使用指南

### 创建任务

1. 在左侧表单中输入任务标题(必填)
2. 选择任务分类和优先级
3. 可选填写描述和预计时长
4. 点击"创建任务"按钮

### 管理任务

- **完成任务**: 点击任务前的复选框
- **编辑任务**: 点击任务卡片上的编辑图标
- **删除任务**: 点击任务卡片上的删除图标

### 查看任务

任务按优先级自动分组显示:
1. 紧急且重要(红色) - 立即处理
2. 重要不紧急(蓝色) - 计划安排
3. 紧急不重要(黄色) - 委托他人
4. 不紧急不重要(灰色) - 稍后处理
5. 已完成(绿色) - 已完成的任务

## 技术栈

- **Vite 6** - 极速构建工具
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **TailwindCSS** - 原子化 CSS
- **本地存储** - 当前使用内存存储(刷新后数据丢失)
